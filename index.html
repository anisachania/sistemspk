3. Detail Perhitungan

Langkah SAW (Simple Additive Weighting)

Rumus Normalisasi: Rij = xij / max(xj) (Benefit) atau min(xj) / xij (Cost)

Langkah WP (Weighted Product)

Rumus: Si = Π (xij ^ wj). Bobot cost menjadi negatif.

Langkah SMART (Simple Multi-Attribute Rating Technique)

Rumus Utility: u(ai) = (Cout - Cmin) / (Cmax - Cmin). SMART mengukur posisi relatif terhadap range data.

5. Perbandingan Visual

Mengapa Hasil SMART Berbeda?

Hasil SMART bisa berbeda dari SAW dan WP karena cara normalisasinya. SAW menormalkan data terhadap nilai maksimal (linear), sedangkan SMART menormalkan data berdasarkan rentang (range) antara nilai minimal dan maksimal.

Jika selisih antara nilai terendah dan tertinggi sangat jauh, SMART akan memberikan skor 0 untuk nilai terendah dan 1 untuk nilai tertinggi, sehingga hasilnya lebih "kontras" dibandingkan SAW.
